# Zoo AI Assistant

This is a backend AI application built using Flask that answers user queries using Google's Gemini API.

## 🚀 Features
- REST API endpoint (/ask) to handle user queries
- AI-generated responses using Gemini API
- Deployed on Google Cloud Run
- Error handling for invalid inputs

## 🛠️ Tech Stack
- Python
- Flask
- Google Gemini API
- Google Cloud Run

## 🌐 Deployment
The application is deployed and publicly accessible via Cloud Run.

## 📌 How it works
User sends a query → API processes request → Gemini generates response → Output returned to user
